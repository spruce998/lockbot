## 1. 项目概述

### 1.1 项目背景

洛克王国世界是一款回合制宠物对战游戏，PVP 对战涉及属性克制、技能选择、状态判断等博弈要素。本项目旨在开发一款自动化辅助工具，帮助用户在闲暇时间实现半自动/全自动 PVP 对战和资源获取。

### 1.2 项目目标

| 阶段 | 目标 | 预计工时 |
|------|------|---------|
| **Phase 1 (MVP)** | 辅助工具：识别对手 + 技能推荐 | 1-2 周 |
| **Phase 2** | 半自动：自动点击技能 + 基础反检测 | 2-4 周 |
| **Phase 3** | 全自动：自动排队 + 跑图 + 资源采集 | 4-8 周 |

### 1.3 使用场景

- 洗衣服、做饭等碎片时间挂机
- 自动完成日常 PVP 任务
- 自动采集地图资源
- 小号养成辅助

### 1.4 约束条件

- **平台**: PC 客户端（Windows）
- **安全**: 图像识别方案，不修改内存/协议
- **风险**: 小号测试，需反检测措施
- **用户**: 有编程经验，可参与开发调试

## 6. 核心算法设计

### 6.1 属性克制计算

```python
def calculate_damage(skill, attacker, defender):
    """
    计算技能伤害
    """
    # 基础伤害
    base_damage = skill.power
    
    # 属性克制倍率
    type_multiplier = type_chart[skill.element].get(defender.element, 1.0)
    
    # 攻击/防御比（Phase 3）
    atk_def_ratio = attacker.attack / defender.defense
    
    # 随机波动 (0.85-1.0)
    random_factor = random.uniform(0.85, 1.0)
    
    # 最终伤害
    damage = base_damage * type_multiplier * atk_def_ratio * random_factor
    
    return damage, type_multiplier
```

### 6.2 技能选择算法（Phase 1: 贪心）

```python
def choose_skill(my_pet, enemy_pet):
    """
    贪心算法：选择期望伤害最高的技能
    """
    best_skill = None
    best_score = -inf
    
    for skill in my_pet.skills:
        # 评分 = 期望伤害 + 先手奖励 - 命中惩罚
        expected_damage = skill.power * skill.accuracy
        type_bonus = get_type_effectiveness(skill.element, enemy_pet.element)
        priority_bonus = skill.priority * 10
        
        score = expected_damage * type_bonus + priority_bonus
        
        if score > best_score:
            best_score = score
            best_skill = skill
    
    return best_skill
```

### 6.3 技能选择算法（Phase 3: Minimax）

```python
def minimax(state, depth, is_maximizing):
    """
    Minimax 算法 + Alpha-Beta 剪枝
    """
    if depth == 0 or state.is_terminal():
        return evaluate_state(state)
    
    if is_maximizing:
        max_eval = -inf
        for action in state.get_actions():
            eval = minimax(state.apply(action), depth - 1, False)
            max_eval = max(max_eval, eval)
        return max_eval
    else:
        min_eval = inf
        for action in state.get_actions():
            eval = minimax(state.apply(action), depth - 1, True)
            min_eval = min(min_eval, eval)
        return min_eval
```

### 6.4 模板匹配算法

```python
def match_template(frame, template, threshold=0.8):
    """
    OpenCV 模板匹配
    """
    result = cv2.matchTemplate(frame, template, cv2.TM_CCOEFF_NORMED)
    min_val, max_val, min_loc, max_loc = cv2.minMaxLoc(result)
    
    if max_val >= threshold:
        h, w = template.shape[:2]
        center = (max_loc[0] + w//2, max_loc[1] + h//2)
        return True, center, max_val
    
    return False, None, max_val
```

### 6.5 鼠标轨迹生成（贝塞尔曲线）

```python
def generate_bezier_trajectory(start, end, control_points, steps=60):
    """
    生成贝塞尔曲线鼠标轨迹
    """
    trajectory = []
    for i in range(steps + 1):
        t = i / steps
        
        # 二次贝塞尔
        x = (1-t)**2 * start[0] + 2*(1-t)*t * control_points[0] + t**2 * end[0]
        y = (1-t)**2 * start[1] + 2*(1-t)*t * control_points[1] + t**2 * end[1]
        
        # 添加噪声
        x += random.uniform(-2, 2)
        y += random.uniform(-2, 2)
        
        trajectory.append((x, y))
    
    return trajectory
```

## 7. 反检测策略

### 7.1 检测手段分析

游戏可能使用的检测手段：

| 检测类型 | 检测内容 | 规避方案 |
|---------|---------|---------|
| **行为检测** | 操作频率固定、反应时间过短 | 随机延迟、拟人化轨迹 |
| **图像检测** | 截图频率固定 | 变频截图、添加噪声 |
| **进程检测** | 已知外挂进程名 | 随机进程名、注入合法进程 |
| **内存检测** | 异常内存读取 | 不使用内存读取方案 |

### 7.2 反检测实现

```python
class AntiDetect:
    def __init__(self):
        self.action_times = []
        self.session_start = datetime.now()
    
    def human_delay(self):
        """随机延迟"""
        time.sleep(random.uniform(0.1, 0.5))
    
    def smooth_move(self, start, end):
        """平滑移动"""
        trajectory = generate_bezier_trajectory(start, end)
        for point in trajectory:
            pyautogui.moveTo(point)
            time.sleep(0.01)
    
    def random_offset(self, x, y):
        """点击偏移"""
        return x + random.randint(-8, 8), y + random.randint(-8, 8)
    
    def should_rest(self):
        """休息检测"""
        elapsed = (datetime.now() - self.session_start).seconds
        if elapsed > random.randint(3600, 7200):
            return True
        return False
```
