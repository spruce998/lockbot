## 1. 对战概述

### 1.1 基本规则

| 项目 | 说明 |
|------|------|
| 战斗类型 | 回合制对战 |
| 胜利条件 | 每位玩家有 **4 格生命值**，击败对方一只精灵扣对方 1 格并恢复自己 1 格，一方生命值归 0 则判负 |
| 每回合行动 | 每回合选择使用技能或切换精灵（二者选一） |
| 技能数量 | 每只精灵**只能携带 4 个技能**上场 |
| 出手顺序 | 同时选择行动，按 **先手值 → 速度** 决定出手顺序 |

### 1.2 技能来源

每只精灵可携带的技能来自三类，可混搭：

1. **自学技能**（`skills` 字段，按等级解锁）
2. **血脉技能**（`bloodlineSkills` 字段，需解锁血脉）
3. **技能石**（`learnableSkillStones` 字段，需装备对应技能石）

## 2. 属性系统

### 2.1 19 种属性

普通、草、火、水、光、地、冰、龙、电、毒、虫、武、翼、萌、幽、恶、机械、幻、无

### 2.2 克制四维

每种属性有四个维度，各分两档：

| 维度 | 普通倍率 | 强力倍率（触发条件） |
|------|---------|---------------------|
| **strong**（攻击克制） | 200% | 300%（目标的两个属性均被攻击属性克制） |
| **resist**（攻击被抵抗） | 50% | 25%（目标的两个属性均抵抗攻击属性） |
| **weak**（防御弱点） | 200% | 300%（自身两个属性均被攻击属性克制） |
| **vulnerable**（防御抵抗） | 50% | 25%（自身两个属性均抵抗攻击属性） |

### 2.3 双属性克制计算

**倍率相乘**。例：
- 草/虫双属性被火打：火克制草(2x) × 火克制虫(2x) = **4 倍伤害**
- 草/虫双属性被火打（强力）：火强克制草(3x) × 火强克制虫(3x) = **9 倍伤害**

## 3. 能量系统

### 3.1 基本规则

| 项目 | 说明 |
|------|------|
| 能量归属 | 能量属于**宠物个体**，换场后保留 |
| 初始能量 | 首次出场获得 **10 点能量** |
| 特殊初始 | 部分精灵特性会改变初始能量，如「布是石」特性地脉：初始 0，入场前己方精灵每放 1 次地系技能回复 3 能量 |
| 能量上限 | 默认 **10**，某些精灵特性可突破上限（需实测探索） |
| 自然恢复 | **每回合不会自然恢复能量** |

### 3.2 能量恢复方式

| 方式 | 说明 |
|------|------|
| **聚能** | 状态技能，回复 5 点能量。但会被克制状态的技能应对（如地刺会打断状态） |
| **技能回复** | 部分技能自带能量回复效果 |
| **应对回复** | 部分应对技能成功后回复能量 |

### 3.3 典型能量回复技能

| 技能 | 能耗 | 效果 |
|------|------|------|
| 聚能 | - | 回复 5 能量（状态技能，可被打断） |
| 抓挠 | 0 | 造成物伤，自己回复 1 能量 |
| 火苗 | 0 | 造成物伤，自己回复 1 能量 |
| 鬼火 | 0 | 造成魔伤，自己回复 1 能量 |
| 魔爪 | 0 | 造成物伤，自己回复 1 能量 |
| 甩水 | 0 | 造成魔伤，自己回复 1 能量 |
| 寸拳 | 0 | 造成物伤，自己回复 1 能量 |
| 虫网 | 0 | 造成魔伤，自己回复 1 能量 |
| 偷师 | 0 | 造成物伤，自己回复 1 能量 |
| 风吹雪 | 0 | 造成魔伤，自己回复 1 能量 |
| 聚盐 | 3 | 2 连击，每次连击回复 5% 生命和 **1 能量**，使用后连击数永久 +1 |
| 徒长 | 2 | 自己回复 **10 能量** |
| 氧输送 | 2 | 自己回复 **4 能量**，并获得魔攻 +80% |

| 洗礼 | 1 | 驱散自己的减益，并获得全技能能耗 -1 |

### 3.4 能量管理策略

```
1. 每回合只能使用一个技能或切换精灵
2. 聚能虽然回复 5 能量，但消耗一个回合且可被打断
3. 优先使用 0 能耗回复技能（抓挠、火苗等）兼顾输出和回复
4. 高能耗技能（5+）需要提前规划能量积攒
5. 部分特性可突破 10 能量上限，需要实测发现
```

## 4. 先后手判定

### 4.1 判定规则

```
先后手 = 先手值 > 速度

1. 先比较技能的先手值，先手值高的一方优先出手
2. 先手值相同时，比较精灵的速度种族值，速度更快的一方先出手
3. 先手+X 效果不可叠加

公式：
先手值 = 技能基础先手值 + buff/debuff 修正
速度值 = 精灵速度种族值 + 性格修正 + buff/debuff 修正
```

### 4.2 影响先后手的技能/效果

| 技能/效果 | 先后手影响 |
|-----------|-----------|
| 天旋地转 | 先手 +1 |
| 先发制人 | 先手 +1 |
| 伪造账单 | 先手 +1 |
| 有效预防 | 应对攻击：下次行动获得先手 +1 |
| 麻痹 | 敌方先手 -1 |
| 迅捷 | 主动更换精灵入场时，按技能排序使用第一个能量满足且带迅捷的技能 |
| 风起印记 | 每层先手攻击时伤害提升 20%（仅 1 层生效） |

### 4.3 速度相关状态

| 状态 | 效果 |
|------|------|
| 减速印记 | 速度 -10（每层） |
| 雪球 | 敌方速度 -90 |
| 丢冰块 | 敌方速度 -30 |
| 乘风 | 自己获得速度 +120 |
| 快速移动 | 速度 +80（应对防御：+160） |
| 超导加速 | 速度 +30 |
| 啮合传递 | 速度 +100（位于 1 号位时能耗 -2，传动） |
| 示弱（萌化） | 速度永久 +150 |
| 开朗/胆小/急躁/天真（性格） | 速度 +10% |

## 8. 战斗流程

```
回合开始
├── 天气效果结算
├── DOT 结算（寄生/冻结/中毒/灼烧）
├── 印记效果结算
│
├── 双方同时选择行动
│   ├── 使用技能（消耗能量）
│   └── 切换精灵（消耗一个回合）
│
├── 判定先后手：先手值 > 速度
│
├── 先手方行动
│   ├── 检查能量是否足够
│   ├── 执行技能效果
│   └── 触发迸发/应对/传动等
│
├── 后手方行动
│   └── 同先手方流程
│
├── 回合结束
│   ├── DOT 结算
│   ├── 印记效果结算
│   └── 检查战斗是否结束
│
└── 进入下一回合
```
