## 1. 战斗系统概述

洛克王国世界采用**回合制**战斗系统，核心规则如下：

| 项目 | 说明 |
|------|------|
| 战斗类型 | 回合制对战（PVP / PVE） |
| 参与方 | 每方携带 1-6 只精灵，战斗中可更换 |
| 每回合行动 | 每回合从 4 个技能中选择 1 个使用 |
| 行动顺序 | 由速度、技能先手度、状态效果共同决定 |
| 战斗结束 | 一方所有精灵血量归零即判负 |
| 能量消耗 | 使用技能消耗能量（非 PP 制） |
| 能量恢复 | 每回合获得固定能量 |

### 1.1 战斗界面元素

| UI 元素 | 作用 |
|---------|------|
| 我方技能栏 | 4 个技能按钮，对应 1-4 号栏位 |
| 我方血条 | 显示当前血量百分比 |
| 我方能量条 | 显示当前能量值 |
| 敌方血条 | 显示敌方血量百分比 |
| 状态图标 | 显示当前 buff / debuff |
| 天气/场地 | 显示全局战斗环境影响 |
| 回合提示 | 显示当前回合数 |

## 2. 能量系统

### 2.1 基本规则

| 项目 | 说明 |
|------|------|
| 初始能量 | 大部分精灵**默认 10 能量**入场 |
| 特殊初始 | 部分精灵有特殊初始能量（如 **幔尾兽 入场 5 能量**） |
| 每回合恢复 | **每回合恢复 2-3 能量**（需实测确认精确值） |
| 技能能耗 | 每个技能有能耗值，范围 **0 ~ 7** |
| 能量上限 | 能量有上限（推测约 10-12，需实测） |
| 能量溢出 | 能量达到上限时，多余的恢复会被浪费 |

### 2.2 特殊初始能量精灵

> 以下为已知特殊初始能量的精灵，需要持续补充。

| 精灵 | 初始能量 | 原因 |
|------|---------|------|
| **幔尾兽** | **5 能量** | 种族/特性差异（需确认具体原因） |
| 大部分精灵 | **10 能量** | 默认值 |

> ⚠️ **需要实测确认**：具体哪些精灵有特殊初始能量，以及精确数值。

### 2.3 能耗等级与示例

| 能耗 | 威力范围 | 示例技能 | 说明 |
|------|---------|---------|------|
| **0** | 0-240 | 抓挠(35)、猛烈撞击(65)、彗星(240) | 0 能耗技能可随意使用，部分 0 能耗有高威力但有代价（如彗星消耗全部生命） |
| **1** | 0-100 | 龙吼(60)、防御(0)、魔法增效(0)、火爪(60) | 低能耗，日常使用频繁 |
| **2** | 0-90 | 力量增效(0)、锐利眼神(0)、扫尾(90) | 中等能耗，需要一定能量管理 |
| **3** | 1-200 | 升龙咆哮(200)、龙炮(100)、破防(0) | 较高能耗，需要积攒能量 |
| **4** | 80-120 | 光刃(120)、藤绞(80)、超维投射(0) | 高能耗，关键时刻使用 |
| **5** | 155 | 湮灭(155) | 极高能耗 |
| **6** | 150 | 吞噬(150) | 极高能耗 |
| **7** | 0-150 | 天洪(150)、润泽(0) | 最高能耗，需要长时间积攒 |

### 2.4 能量回复技能

| 技能 | 效果 |
|------|------|
| **徒长** | 自己回复 10 能量 |
| **蓄水** | 下次使用的技能能耗 -6 |
| **假寐** | 自己回复 2 能量，己方队伍获得 1 次奉献：能耗 -2 |
| **迸发机制** | 部分技能有"迸发"效果，本次技能能耗 -1 |
| **吞噬** | 若使用本技能击败敌方，回复 6 能量 |

### 2.5 能量管理原则

```
基本原则：
1. 保持能量在 2-4 范围内（可灵活使用大多数技能）
2. 避免能量溢出（满能量时恢复被浪费）
3. 高能耗技能（5-7）需要提前积攒
4. 0-1 能耗技能可随时使用，但威力通常较低
5. 能量回复技能是关键转折点，合理使用可以逆转局势
6. 注意部分精灵初始能量较低（如幔尾兽 5 能量），需要额外规划
```

## 3. 先后手与速度

### 3.1 行动顺序决定因素

```
┌─────────────────────────────────────────────────────────────┐
│                先后手判定优先级                               │
│                                                             │
│  1️⃣ 技能先手度（priority）                                    │
│     先手度高的技能优先出手                                     │
│     示例：天旋地转 "先手+1"                                   │
│                                                             │
│  2️⃣ 精灵速度种族值 + 性格修正                                 │
│     先手度相同时，速度更高的精灵优先                            │
│     速度种族值范围：约 45-120                                 │
│                                                             │
│  3️⃣ 状态效果影响                                             │
│     - 麻痹/冻结等状态可能使精灵无法行动或延后                   │
│     - 迅捷状态获得先手优势                                    │
│     - "应对防御"可能改变行动顺序                               │
│                                                             │
│  4️⃣ 随机因素                                                 │
│     速度完全相同时可能随机判定                                 │
└─────────────────────────────────────────────────────────────┘
```

### 3.2 影响先后手的技能/效果

| 技能/效果 | 先后手影响 |
|-----------|-----------|
| 天旋地转 | 先手 +1 |
| 有效预防 | 应对攻击时：下次行动获得先手 +1 |
| 麻痹 | 敌方先手 -1 |
| 迅捷 | 获得先手优势 |
| 超导加速 | 速度 +80 |
| 快速移动 | 速度 +80（应对防御：速度 +160） |
| 羽化加速 | 全技能威力 +20，迅捷 |

### 3.3 速度种族值参考

| 精灵 | 速度种族值 | 特点 |
|------|-----------|------|
| 阿米亚特 | 45 | 极慢 |
| 阿米樱 | 60 | 慢 |
| 阿布 | 80 | 中等 |
| 矮脚爬爬 | 92 | 偏快 |
| 迪莫 | 92 | 偏快 |
| （高速精灵） | 100+ | 快 |

### 3.4 应对/防御机制

```
应对（Counter）机制：
- 部分技能有"应对攻击"或"应对状态"效果
- 当对手使用特定类型技能时，触发额外效果
- 可能改变行动顺序或技能效果

示例：
- 防御: "减伤 70%，应对攻击"
- 有效预防: "减伤 50%，应对攻击：下一次行动获得先手+1"
- 抽枝: "应对状态：自己回复 50%生命和 5 能量"
- 快速移动: "应对防御：改为速度+160"
```

### 4.4 双属性克制计算

```
双属性精灵的克制倍率计算：
最终倍率 = 对属性 A 的倍率 × 对属性 B 的倍率

示例：
- 某精灵为 草+毒 双属性
- 使用 火系技能攻击：
  - 火对草 = 2.0
  - 火对毒 = 1.0
  - 最终 = 2.0 × 1.0 = 2.0 倍

- 某精灵为 草+水 双属性
- 使用 电系技能攻击：
  - 电对草 = 0.5
  - 电对水 = 2.0
  - 最终 = 0.5 × 2.0 = 1.0 倍（正常）

> ⚠️ 需要游戏内验证双属性是否存在，以及计算方式。
```

## 6. 技能位置效果

### 6.1 基本规则

| 项目 | 说明 |
|------|------|
| 技能栏位 | 每只精灵最多携带 **4 个技能**，放置在 **1-4 号栏位** |
| 位置效果 | 部分技能在不同栏位有不同效果 |
| 栏位顺序 | 1 号 → 2 号 → 3 号 → 4 号（从上到下） |

### 6.2 已知位置效果

| 技能 | 位置效果 |
|------|---------|
| **离子震荡** | 位于 3 号位时威力 +40，传动 1 |
| **啮合传递** | 位于 1 号或 3 号位时额外获得物攻 +60%，传动 1 |
| **折射** | 携带其他系别技能会给本技能带来不同效果 |
| **过曝** | 每使用过 1 个其他系别技能，本技能威力永久 +30 |
| **漫反射** | 每种系别中的至多 1 个技能，威力 +35 |
| **放晴** | 光系技能威力永久 +50%（应对防御：改为永久 +100%） |

### 6.3 位置策略

```
技能摆放策略：
1. 位置效果技能放在指定栏位（如离子震荡放 3 号位）
2. 不同系别技能分散摆放可以触发"漫反射"等效果
3. 折射技能的效果取决于携带的其他系别技能
4. 需要权衡技能效果和能量消耗
```

## 7. 特殊机制

### 7.1 蓄力（Charge）

| 技能 | 效果 |
|------|------|
| 升龙咆哮 | 蓄力，对敌方造成魔法伤害（200 威力） |
| 龙之利爪 | 蓄力，造成物伤并吸血 50%（130 威力） |
| 怨力打击 | 蓄力，若蓄力期间受到攻击，威力变为敌方技能威力的 3 倍 |
| 架势 | 回复 20% 生命，下次技能无需蓄力 |

```
蓄力机制：
- 蓄力技能通常需要 1 回合准备
- 准备完成后下回合释放
- 部分技能（架势）可以跳过蓄力
- 蓄力期间受到攻击可能改变效果（如怨力打击）
```

```
迸发机制：
- 满足特定条件时触发额外效果

- 需要在决策时考虑是否触发迸发
```

```
传动机制：

- 可能传递能量、增益或其他效果
- 在队伍搭配时需要考虑传动配合
```

## 10. 伤害计算

### 10.1 伤害计算因素

```
伤害 = 基础威力 × 属性克制倍率 × (攻击方种族值/防御方种族值) 
     × 随机波动(0.85-1.0) × 状态修正 × 位置效果加成

影响因素：
1. 技能基础威力（0-240）
2. 属性克制倍率（0x, 0.5x, 1x, 2x）
3. 攻击方攻击种族值（物攻或魔攻）
4. 防御方防御种族值（物防或魔防）
5. 随机波动（0.85-1.0）
6. 增益/减益状态
7. 技能位置效果
8. 天气/场地效果
9. 特性加成
```

### 10.2 种族值与伤害关系

| 精灵 | 物攻 | 魔攻 | 物防 | 魔防 | 说明 |
|------|------|------|------|------|------|
| 阿布 | 68 | 68 | 83 | 83 | 平衡型 |
| 阿米亚特 | 95 | 47 | 67 | 42 | 物攻型 |
| 阿米樱 | 127 | 62 | 89 | 56 | 高物攻型 |
| 矮脚爬爬 | 117 | 110 | 74 | 59 | 双攻型 |
| 暗影灵面 | 30 | 96 | 83 | 88 | 魔攻型 |

## 11. 换宠机制

### 11.1 基本规则

| 项目 | 说明 |
|------|------|
| 携带数量 | 每方可携带多只精灵（通常 6 只） |

| 换宠收益 | 利用属性克制，躲避不利局面 |

### 11.2 换宠策略

```
换宠决策：
1. 当我方精灵处于劣势时换宠
2. 利用属性克制选择有利精灵
3. 预测对手行动，提前换上克制精灵
4. 考虑换宠后能量状态（新精灵可能有不同初始能量）
5. 队伍中精灵搭配需要考虑属性覆盖
```

## 13. 战斗流程总结

```
┌─────────────────────────────────────────────────────────────┐
│                    单回合战斗流程                             │
│                                                             │
│  1. 回合开始                                                 │
│     - 双方获得能量（约 2-3 点）                               │
│     - 持续伤害结算（灼烧/冻结/中毒等）                        │
│     - 天气/场地效果生效                                       │
│                                                             │
│  2. 双方选择技能                                             │
│     - 从 4 个技能栏位中选择 1 个                              │
│     - 考虑能量、克制、位置效果等因素                          │
│                                                             │
│  3. 判定先后手                                               │
│     - 技能先手度 → 速度种族值 → 状态效果 → 随机               │
│                                                             │
│  4. 先手方行动                                               │
│     - 检查能量是否足够                                        │
│     - 执行技能效果（伤害/状态/回复等）                        │
│     - 触发迸发/应对/传动等特殊机制                            │
│     - 消耗对应能量                                            │
│                                                             │
│  5. 后手方行动                                               │
│     - 同先手方流程                                            │
│                                                             │
│  6. 回合结束                                                 │
│     - 检查是否有精灵血量归零                                  │
│     - 检查是否战斗结束                                        │
│     - 永久加成更新（如过曝叠加）                              │
│                                                             │
│  7. 进入下一回合                                             │
└─────────────────────────────────────────────────────────────┘
```

## 14. PVP 策略要点

### 14.1 能量管理

| 策略 | 说明 |
|------|------|
| 保持能量 2-4 | 可灵活使用大多数技能 |
| 避免溢出 | 满能量时恢复被浪费 |
| 积攒高能耗 | 提前规划使用 5-7 能耗技能 |
| 善用回复技 | 徒长/蓄水/假寐是关键转折 |
| 注意初始能量 | 低初始能量精灵（如幔尾兽 5 能量）需提前规划 |

### 14.2 属性克制

| 策略 | 说明 |
|------|------|
| 优先克制 | 选择克制属性的技能造成 2x 伤害 |
| 避开抵抗 | 避免使用被抵抗属性的技能 |
| 双属性考量 | 双属性精灵的克制倍率需要叠加计算 |
| 换宠克制 | 利用换宠切换有利属性 |

### 14.3 技能搭配

| 策略 | 说明 |
|------|------|
| 位置效果 | 将技能放在正确栏位（如离子震荡放 3 号位） |
| 系别搭配 | 不同系别技能可以触发"漫反射"等效果 |
| 蓄力配合 | 架势可以跳过蓄力，提高蓄力技能灵活性 |
| 能量平衡 | 高低能耗技能搭配使用 |

### 14.4 状态管理

| 策略 | 说明 |
|------|------|
| 施加 DOT | 灼烧/冻结/中毒持续造成伤害 |
| 增益叠加 | 物攻/魔攻增益提高伤害输出 |
| 减益对手 | 降低敌方双防提高伤害 |
| 应对防御 | 预测对手行动触发应对效果 |

### 14.5 特性利用

| 策略 | 说明 |
|------|------|
| 克制触发 | 迪莫造成克制伤害后获得增益 |
| 蓄力优化 | 超聚能改变蓄力机制 |
| 特性搭配 | 根据特性选择合适的技能组合 |
