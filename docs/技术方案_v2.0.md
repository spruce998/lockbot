## 1. 项目概述

### 1.1 项目背景

洛克王国世界是一款回合制宠物对战游戏，PVP 对战涉及属性克制、技能选择、状态判断等复杂博弈要素。本项目旨在开发一款自动化辅助工具，从**辅助推荐**逐步迭代到**半自动对战**再到**全自动挂机**，同时支持跑图、资源采集等功能。

### 1.2 项目目标

| 阶段 | 目标 | 交付周期 |
|------|------|---------|
| **Phase 1 (MVP)** | 辅助工具：识别对手 + 技能推荐 | 1-2 周 |
| **Phase 2** | 半自动：自动点击技能 + 反检测增强 | 2-4 周 |
| **Phase 3** | 全自动：PVP + 跑图 + 资源采集 | 4-8 周 |
| **Phase 4+** | 高级 AI 决策、多账号管理、云端同步 | 8-16 周 |

### 1.3 核心设计原则

| 原则 | 说明 |
|------|------|
| **模块化** | 各功能模块独立，可插拔 |
| **扩展性** | 预留接口，方便后续功能扩展 |
| **安全性** | 图像识别方案，不修改游戏内存/协议 |
| **可配置** | 所有参数通过配置文件调整，无需改代码 |
| **可观测** | 完善的日志、监控、统计功能 |

### 2.3 事件驱动架构

系统采用**事件驱动**设计，各模块通过事件总线通信，实现解耦：

```python
# 事件定义示例
class Event(Enum):
    # 战斗事件
    BATTLE_START = "battle.start"
    BATTLE_END = "battle.end"
    TURN_START = "turn.start"
    SKILL_USED = "skill.used"
    PET_SWITCHED = "pet.switched"
    
    # 识别事件
    PET_IDENTIFIED = "pet.identified"
    SKILL_IDENTIFIED = "skill.identified"
    HP_CHANGED = "hp.changed"
    
    # 系统事件
    SYSTEM_START = "system.start"
    SYSTEM_STOP = "system.stop"
    ERROR_OCCURRED = "error.occurred"

# 事件总线使用示例
event_bus.subscribe("battle.start", on_battle_start)
event_bus.subscribe("pet.identified", on_pet_identified)
event_bus.publish("battle.start", enemy_pet=pet_name)
```

**扩展性优势**：
- 新增功能只需订阅对应事件，无需修改现有代码
- 模块间零耦合，可独立测试
- 支持事件录制/回放，方便调试

### 3.2 详细表结构

| `name_cn` | VARCHAR(50) | - | NULL | 中文名（可选） |

| `sub_element` | VARCHAR(20) | - | NULL | 副属性（双属性宠物） |
| `base_hp` | INTEGER | NOT NULL | 100 | 基础血量 |
| `base_atk` | INTEGER | - | 50 | 基础攻击力 |
| `base_def` | INTEGER | - | 50 | 基础防御力 |
| `base_spd` | INTEGER | - | 50 | 基础速度 |
| `base_spatk` | INTEGER | - | 50 | 基础特攻 |
| `base_spdef` | INTEGER | - | 50 | 基础特防 |
| `rarity` | VARCHAR(20) | - | 'common' | 稀有度（common/uncommon/rare/epic/legendary） |

#### 4.1.3 状态机设计

```python
from enum import Enum, auto

class BattleState(Enum):
    """战斗状态枚举"""
    IDLE = auto()           # 空闲状态
    DETECTING = auto()      # 检测战斗中
    BATTLE_START = auto()   # 战斗开始
    IDENTIFYING = auto()    # 识别敌方
    DECIDING = auto()       # 决策中
    EXECUTING = auto()      # 执行操作
    WAITING = auto()        # 等待回合切换
    BATTLE_END = auto()     # 战斗结束
    ERROR = auto()          # 错误状态


class PVPStateMachine:
    """PVP 状态机"""
    
    def __init__(self):
        self.state = BattleState.IDLE
        self.transitions = {
            BattleState.IDLE: {
                "detect_battle": BattleState.DETECTING,
            },
            BattleState.DETECTING: {
                "battle_confirmed": BattleState.BATTLE_START,
                "no_battle": BattleState.IDLE,
            },
            BattleState.BATTLE_START: {
                "start_identify": BattleState.IDENTIFYING,
            },
            BattleState.IDENTIFYING: {
                "identified": BattleState.DECIDING,
                "identify_failed": BattleState.ERROR,
            },
            BattleState.DECIDING: {
                "decision_made": BattleState.EXECUTING,
                "decision_timeout": BattleState.ERROR,
            },
            BattleState.EXECUTING: {
                "executed": BattleState.WAITING,
                "execution_failed": BattleState.ERROR,
            },
            BattleState.WAITING: {
                "turn_changed": BattleState.DECIDING,
                "battle_ended": BattleState.BATTLE_END,
            },
            BattleState.BATTLE_END: {
                "logged": BattleState.IDLE,
            },
            BattleState.ERROR: {
                "recovered": BattleState.IDLE,
                "fatal": BattleState.IDLE,  # 严重错误也回到空闲
            },
        }
    
    def transition(self, event: str) -> bool:
        """执行状态转换"""
        if event in self.transitions.get(self.state, {}):
            old_state = self.state
            self.state = self.transitions[self.state][event]
            self._on_state_change(old_state, self.state)
            return True
        return False
    
    def _on_state_change(self, old: BattleState, new: BattleState):
        """状态变更回调"""
        logger.info(f"状态变更：{old.name} -> {new.name}")
        event_bus.publish(f"state.{new.name.lower()}", old_state=old, new_state=new)
```

```python
class DecisionEngine:
    """
    PVP 决策引擎
    
    支持多种决策策略，可通过配置切换：
    - greedy: 贪心算法（Phase 1）
    - minimax: Minimax + Alpha-Beta 剪枝（Phase 2）
    - mcts: 蒙特卡洛树搜索（Phase 3）
    - ml: 机器学习模型（Phase 4）
    """
    
    def __init__(self, strategy: str = "greedy", db: Database = None):
        self.strategy = strategy
        self.db = db
        self.strategies = {
            "greedy": GreedyStrategy(),
            "minimax": MinimaxStrategy(depth=3),
            "mcts": MCTSStrategy(simulations=1000),
            "ml": MLStrategy(model_path="models/pvp_model.pkl"),
        }
    
    def decide(self, context: BattleContext) -> Decision:
        """
        做出决策
        
        :param context: 当前战斗上下文
        :return: 决策结果
        """
        strategy = self.strategies.get(self.strategy)
        if not strategy:
            raise ValueError(f"未知策略：{self.strategy}")
        
        return strategy.execute(context)
```

```python
class GreedyStrategy:
    """贪心策略：选择期望伤害最高的技能"""
    
    def execute(self, ctx: BattleContext) -> Decision:
        best_skill = None
        best_score = float('-inf')
        
        for skill in ctx.my_pet.skills:
            # 计算有效伤害
            type_mult = self._get_type_multiplier(skill.element, ctx.enemy.element)
            expected_dmg = skill.power * skill.accuracy * type_mult
            
            # 先手度加分
            priority_bonus = skill.priority * 10
            
            # 低血量时优先攻击技能
            if ctx.my_pet.hp_percent < 0.3:
                if skill.effect_type == 'damage':
                    expected_dmg *= 1.2
            
            score = expected_dmg + priority_bonus
            
            if score > best_score:
                best_score = score
                best_skill = skill
        
        return Decision(
            action="use_skill",
            skill=best_skill,
            confidence=min(1.0, best_score / 200),
            reason=f"期望伤害 {best_score:.1f}"
        )
```

```python
class ImagePreprocessor:
    """图像预处理管道"""
    
    def __init__(self, config: dict = None):
        self.config = config or {}
        self.pipeline = []
        
        # 根据配置构建预处理管道
        if self.config.get("grayscale", True):
            self.pipeline.append(self._to_grayscale)
        if self.config.get("denoise", True):
            self.pipeline.append(self._denoise)
        if self.config.get("enhance_contrast", False):
            self.pipeline.append(self._enhance_contrast)
        if self.config.get("threshold", False):
            self.pipeline.append(self._threshold)
    
    def process(self, image: np.ndarray) -> np.ndarray:
        """执行预处理管道"""
        result = image.copy()
        for step in self.pipeline:
            result = step(result)
        return result
    
    def _to_grayscale(self, image):
        return cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
    
    def _denoise(self, image):
        return cv2.fastNlMeansDenoising(image)
    
    def _enhance_contrast(self, image):
        clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8))
        return clahe.apply(image)
    
    def _threshold(self, image):
        _, result = cv2.threshold(image, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
        return result
```

```python
    def __init__(self, db: Database):
        self.db = db
        self._cache = {}
        self._load_from_db()
    
    def get(self, key: str, default=None):
        """获取配置值"""
        if key not in self._cache:
            return default
        
        setting = self._cache[key]
        return self._parse_value(setting.value, setting.value_type)
    
    def set(self, key: str, value):
        """设置配置值"""
        setting = self._cache.get(key)
        if not setting:
            raise KeyError(f"配置项不存在：{key}")
        
        setting.value = str(value)
        setting.updated_at = datetime.now()
        self.db.update_setting(setting)
        
        # 发布配置变更事件
        event_bus.publish("config.changed", key=key, value=value)
    
    def get_category(self, category: str) -> dict:
        """获取某分类下的所有配置"""
        return {
            k: v for k, v in self._cache.items()
            if v.category == category
        }
```

```python
class BehaviorSimulator:
    """
    行为模拟器
    模拟人类操作的不规则性
    """
    
    def __init__(self, profile: str = "normal"):
        """
        :param profile: 行为画像
            - cautious: 谨慎（延迟大、频率低）
            - normal: 正常
            - aggressive: 激进（延迟小、频率高，风险大）
        """
        self.profiles = {
            "cautious": {"delay_range": (0.3, 1.0), "error_rate": 0.05},
            "normal": {"delay_range": (0.1, 0.5), "error_rate": 0.02},
            "aggressive": {"delay_range": (0.05, 0.2), "error_rate": 0.01},
        }
        self.profile = self.profiles[profile]
    
    def add_reaction_delay(self):
        """模拟人类反应时间"""
        delay = random.uniform(*self.profile["delay_range"])
        time.sleep(delay)
    
    def simulate_hesitation(self):
        """模拟犹豫（偶尔改变技能选择）"""
        if random.random() < self.profile["error_rate"]:
            # 模拟选错技能后立即纠正
            time.sleep(random.uniform(0.2, 0.5))
            return True
        return False
    
    def mouse_jitter(self, duration: float = 1.0):
        """鼠标微小抖动（模拟无意识动作）"""
        start_x, start_y = pyautogui.position()
        steps = int(duration * 60)
        
        for _ in range(steps):
            offset_x = random.uniform(-3, 3)
            offset_y = random.uniform(-3, 3)
            pyautogui.moveRel(offset_x, offset_y, duration=duration/steps)
            time.sleep(duration / steps)
        
        # 移回原位
        pyautogui.moveTo(start_x, start_y, duration=0.2)
```

### 5.4 时间管理器

```python
class TimeManager:
    """
    时间管理器
    控制运行时段、休息间隔等
    """
    
    def __init__(self, config: dict):
        self.config = config
        self.session_start = datetime.now()
        self.last_rest = datetime.now()
    
    def should_run(self) -> bool:
        """判断当前是否应该运行"""
        now = datetime.now()
        hour = now.hour
        
        # 检查活跃时段
        start = self.config.get("active_hours_start", 8)
        end = self.config.get("active_hours_end", 23)
        
        if not (start <= hour <= end):
            return False
        
        # 检查是否需要休息
        if self._should_rest():
            return False
        
        return True
    
    def _should_rest(self) -> bool:
        """判断是否需要休息"""
        elapsed = (datetime.now() - self.last_rest).total_seconds()
        min_interval = self.config.get("rest_interval_min", 60) * 60
        max_interval = self.config.get("rest_interval_max", 120) * 60
        
        if elapsed < min_interval:
            return False
        
        if elapsed > max_interval:
            return True
        
        # 在区间内随机决定是否休息
        return random.random() < 0.3
    
    def get_rest_duration(self) -> int:
        """获取休息时长（分钟）"""
        min_rest = self.config.get("rest_duration_min", 5)
        max_rest = self.config.get("rest_duration_max", 15)
        return random.randint(min_rest, max_rest)
```

系统设计为**插件化架构**，新功能可通过插件方式接入：

```python
# 插件基类
class Plugin(ABC):
    """插件基类"""
    
    @abstractmethod
    def name(self) -> str:
        """插件名称"""
        pass
    
    @abstractmethod
    def version(self) -> str:
        """插件版本"""
        pass
    
    @abstractmethod
    def initialize(self, context: PluginContext):
        """初始化插件"""
        pass
    
    @abstractmethod
    def destroy(self):
        """销毁插件"""
        pass
    
    def enabled(self) -> bool:
        """是否启用"""
        return True


# 插件管理器
class PluginManager:
    """插件管理器"""
    
    def __init__(self):
        self.plugins: Dict[str, Plugin] = {}
    
    def register(self, plugin: Plugin):
        """注册插件"""
        if plugin.name() in self.plugins:
            raise ValueError(f"插件已存在：{plugin.name()}")
        self.plugins[plugin.name()] = plugin
    
    def initialize_all(self, context: PluginContext):
        """初始化所有插件"""
        for plugin in self.plugins.values():
            if plugin.enabled():
                plugin.initialize(context)
    
    def get_plugin(self, name: str) -> Optional[Plugin]:
        """获取插件"""
        return self.plugins.get(name)
```

```python
class Recognizer(ABC):
    """识别器接口"""
    
    @abstractmethod
    def recognize(self, image: np.ndarray) -> List[RecognitionResult]:
        """识别图像"""
        pass
    
    @abstractmethod
    def get_type(self) -> str:
        """识别器类型"""
        pass


# 实现多种识别器
class TemplateMatcher(Recognizer): ...
class OcrRecognizer(Recognizer): ...
class YoloDetector(Recognizer): ...
class CustomRecognizer(Recognizer): ...

# 识别器链
class RecognizerChain:
    def __init__(self):
        self.recognizers: List[Recognizer] = []
    
    def add(self, recognizer: Recognizer):
        self.recognizers.append(recognizer)
    
    def recognize(self, image: np.ndarray) -> RecognitionResult:
        for recognizer in self.recognizers:
            results = recognizer.recognize(image)
            if results:
                return results[0]
        return None
```
